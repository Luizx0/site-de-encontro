# Site de Encontro

Um convite interativo com visual cinematográfico, inspirando Michael Jackson, Stray Kids, Star Wars, Black Clover e Interestellar.

## Como usar

1. Instale dependências: `npm install`
2. Rode em desenvolvimento: `npm run dev`
3. Gere a build: `npm run build`

## Deploy no GitHub Pages

1. Adicione o repositório ao GitHub.
2. Instale dependências: `npm install`
3. Rode: `npm run deploy`

O comando `deploy` usa `gh-pages` para publicar a pasta `dist` no branch `gh-pages`.
